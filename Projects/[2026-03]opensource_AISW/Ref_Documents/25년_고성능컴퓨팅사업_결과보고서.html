□ 과제 개요 및 필요성 

 

  ㅇ(개요)  

본 과제는 기존 트랜스포머 기반 자기회귀(Autoregressive) 모델의 구조적·비용적 한계를 극복하고, 고속 추론·고효율 학습이 가능한 차세대 한국형 파운데이션 모델을 개발하는 것을 목적으로 함. 

특히 확산모델(Diffusion LLM)과 강화사전학습(Reinforcement Pre-training, RPT)을 결합하여 한국어 중심의 차세대 언어모델 구조를 정립하고, 데이터 효율성과 추론 성능을 동시에 만족하는 새로운 학습 방법론을 구축하고자 함. 

● 확산 기반 언어모델의 사전학습 체계 정립 및 모델 구조 비교 검증 

● Block Diffusion 기반 한국어 특화 LLM 개발 및 고속 추론 성능 확보 

● 강화사전학습(RPT) 모듈 설계·구현을 통한 reasoning·대화·코딩 등 고차원 태스크의 성능 향상 

● 학습 파이프라인·데이터셋·코드 공개를 통한 국내 AI 생태계 확산 기반 마련 

 

  ㅇ(필요성)  

최근 Agentic AI의 확산과 함께 고속·저비용·실시간 처리가 가능한 LLM의 수요가 급증하고 있으나, 기존 초거대 모델 기반 방식은 연산 자원·비용·데이터 측면에서 국내 환경에 적합하지 않음. 

또한 한국어 중심 데이터와 연산 자원이 부족한 상황에서 글로벌 모델 의존도가 심화되고 있어 한국형 차세대 모델 구조 개발의 필요성이 매우 높음. 

● 기존 AR 모델의 구조적 한계(느린 추론, 높은 비용, 대규모 데이터 필요성)를 극복할 확산 기반 LLM의 도입 필요 

● 국내 연산 자원 제약 상황에서도 학습 가능한 고효율 모델 및 학습 방법론(RPT, block diffusion) 개발 필요 

● 에이전트·검색·모바일·공공서비스 등 한국어 중심 환경에서 사용할 국산 파운데이션 모델 확보의 전략적 중요성 

● 향후 멀티모달 확장(any-to-any) 및 온디바이스 활용을 위한 새로운 모델 패러다임 정립 필요

   - Diffusion Model 학습 방법론 검증 

본 연구팀은 기존 자기회귀(Autoregressive) 기반 언어모델의 구조적 한계를 극복하고, 고속 추론과 데이터 효율성 극대화를 달성하기 위해 확산 기반 언어모델(Diffusion LLM)의 사전학습 방법론 및 구조 검증을 수행하였음. 

다음과 같은 세부 실험을 진행함: 

 

◆ From-scratch Diffusion 학습 

약 2T 규모의 사전학습 데이터를 활용해 토큰 단위 확산 모델을 처음부터 학습하는 방법으로 가장 전통적인 모델 학습 방식이며 최초부터 확산 모델의 형태를 갖추고 학습했다는 점에서 가장 근본적인 형태임. 그러나 많은 데이터를 필요로 하며 학습에 많은 연산이 필요하다는 점, 데이터의 조합과 형태에 따라 성능에 큰 편차를 보일 수 있으며 그 학습 한계치에 대한 판단이 학습 전에는 어렵다는 단점이 있음.  

 

◆ AR → Diffusion 전이학습 

기존에 존재하던 자기회귀 모델을 기준으로 확산 모델의 디코딩 방식을 추가적으로 학습하는 생성모델 구축 방법임. 다수의 최근 연구가 이 방식을 채택하고 있으며 기존 모델이 갖고 있던 지식층을 충분히 활용한다는 점에서 데이터 활용에 효율적임.  

 

◆ AR → Block Diffusion 학습 

확산 모델의 디코딩을 블록 단위로 구분해서 순차별로 진행하는 하이브리드 형식의 확산모델 학습 방법론임. 순차적인 생성을 통해 자기회귀 모델이 갖는 기본적인 다음토큰예측(Next token prediction)과 유사한 형태의 추론을 통해 더욱 효과적으로 디코딩이 가능하다는 점과 함께 학습에 필요하는 데이터의 숫자가 현저히 줄어든다는 장점을 갖고 있음. 또한 디코딩해야하는 단어의 숫자가 줄어든다는 점에서 추론시의 속도 증가에도 크게 장점이 있는 방법론임.  

 

그림2 에서 볼 수 있듯, 학습 과정을 같은 환경에서 실험한 결과 세 방법론 모두 빠르게 안정화되고 수렴화해가는 과정을 발견할 수 있으나, 도달하는 최종 수치, 그리고 수렴에 도달하는 속도를 모두 비교하였을때 현저한 차이가 있다는 것을 확인할 수 있음.  

이러한 검증 결과를 토대로 본 과제에서는 Block Diffusion LLM이 한국어 차세대 모델 학습에 적합한 구조임을 확인하고, 이후의 모든 모델 구축 단계에서 해당 구조를 채택함. 

 

그림 2 학습방법론 학습 과정별 loss 값 변화 

 

 

 

 

   - 국내 최초 Diffusion Large Language Model 제작 

본 과제는 국내 최초로 한국어 중심 Diffusion 기반 대규모 언어모델을 설계·제작하는 것을 목표로 하였으며, 다음의 성과를 달성함: 

◆ 한국어 데이터에 특화된 최초의 Diffusion LLM 사전학습 착수 

자체 확보 텍스트 2T(Tokens)와 약 100B 규모의 Diffusion 사전학습용 한국어 정제 데이터를 확보하여 사전학습에 활용할 수 있게 됨. 최신 모델들 기준 10T(Tokens) 이상을 필요로하는 기존 AR 모델 대비 데이터 효율성을 크게 향상함.  

◆ Block Diffusion Large Language Model 개발 성공 

생성하는 전체 텍스트를 동시에 denoising 하는 기존의 Diffusion 대비 성능·scalability 가 우수한 방법론을 적용함. 자기회귀(AR) 모델과 확산(Diffusion)모델의 장점을 접합한 하이브리드 방법론으로 특히 적은 데이터로도 학습 효능이 뛰어나 한국어 데이터가 비교적 적은 reasoning, math, code 와 같은 분야에서도 빠른 수렴 특성을 확인함.  

◆ 한국어·일본어 벤치마크에서 초기 성능 산출 

KMMLU, KoBEST, HRM8K 등에서 AR 기반 7B 모델들과 유사한 성능 확보함. 이는 동일 토큰 수 대비 확산 모델의 데이터 효율성이 AR 대비 높음을 의미함 

◆ 고속 추론 실험(Inference Speed) 진행 

Batch size 1·4 기준, Qwen2.5(7B) 규모에서 2배 이상의 출력속도(Tokens per Second)를 확인함. 문제 난이도 및 답변 길이에 따라 출력속도의 편차가 존재한다는 점을 발견할 수 있었고, 추가적인 출력속도의 증가가 가능함을 확인함.  

 

본 과제를 통해 확산 모델의 다방면적인 개발에 필요한 기술들을 검증, 실증하며 한국어 특화 Diffusion LLM의 실효성과 가능성을 입증했으며, 이는 국내 AI 생태계에서 최초이자 중요한 기술적 진전으로 LLM 기술의 활용처의 확대로 이어질 것임. 

 

- Diffusion LLM 학습 방법론 고도화 

더 강력한 모델의 구성을 위해 Mid-training과 Reinforcement Learning(RL) 기반의 확장 기법을 설계해 실험함. 

 

(1) Mid-training 

기존 사전학습된 AR 혹은 Diffusion 모델이 가지는 지식 퇴화(forgetting) 문제를 해결하고, reasoning·math·coding 등 고차원 태스크 성능을 향상하기 위해 mid-training 전략을 도입함 

구체적으로는: 

● Reasoning 특화 데이터(Nemotron post-training data 등) 활용 

Fast-dLLM-v2 baseline에서 유효성이 검증된 데이터 

● Block size warm-up 전략 

Block size를 2→4→8→16→32로 점진적으로 상승 

AR 모델에서 발생하는 forgetting을 완화 

Block Diffusion의 수렴 안정성 및 context 유지 능력을 개선 

 

(2) RL (RPT: Reinforcement Pre-training) 

본 연구는 Diffusion 기반 RL 사전학습(RPT, Reinforcement Pre-training)을 구축하는 것을 목표로 함. 

중간보고서 기준 다음과 같은 기법이 설계됨: 

● Diffusion 복원 과정 전체를 ‘정책(policy)’로 간주한 RL 구조 설계 

● 각 timestep(복원 단계)을 연속적 의사결정으로 모델링 

● 정책 경사(policy gradient) 기반 보상 업데이트 수행 

보상체계: 

● 정답 검증 기반 보상(자동화) 

● 인간 선호 기반 보상(Human Preference Reward) 

● 대화, 글쓰기, 문장 품질 등 주관적 태스크에 적용 

● 이중 보상 시스템을 통해 모델의 reasoning·창작 능력 모두 강화 

이러한 RL 기반 diffusion 학습은 본 연구가 국내 최초로 시도하는 형태이며, 향후 RPT 를 학습에 전면적으로 활용할 수 있는 기술로써 최종적으로 총집될 예정임.

-일반(영문) 성능 지표 

Trida-7B-preview 모델은 본 과제를 통해 최초로 개발된 한국어 중심 Diffusion 기반 언어모델로서, 여러 주요 벤치마크(GSM8K, MMLU, GPQA, Ifeval)에 대해 기존 Diffusion 기반 모델들과 비교·평가하여 의미있는 초기 성과를 도출하였음. 

우선, MMLU 및 Ifeval과 같이 지식 기반 이해 능력 및 문장 생성 품질을 평가하는 항목에서 타 모델 대비 우수한 성능을 확보함으로써, Diffusion 구조가 언어 이해·문장 품질·정합성 면에서 충분한 경쟁력을 확보할 있는 기술적 발전을 이룩함. 특히 문맥을 전체적으로 고려하는 Diffusion 특유의 복원 과정에서 문장 안정성과 자연스러움을 향상시킬 수 있었음. 

반면, 수학적 연산과 절차적 추론을 요구하는 일부 과제에서는 상대적으로 낮은 성능을 보였으나, 이는 해당 영역의 데이터 구성과 적용이 초기 단계임을 감안할 때 자연스러운 결과로 평가됨. 특히 기존 연구 사례에서도 reasoning 중심의 성능은 사후학습·보상 모델 적용을 통해 크게 향상되는 경향이 보고되고 있어, 향후 RL 기반 정교화 및 reasoning-specialized 데이터 투입 시 가장 큰 개선 폭이 기대되는 영역임. 

 

종합적으로 Trida-7B-preview는 초기 버전임에도 불구하고 다음과 같은 요소들에 대한 성과를 이룩함 

● 지식 기반 성능에서 경쟁력 확보 

● 언어 생성 품질에서 우수한 결과 확인 

● 추론 능력의 개선

- 다국어(한국어 외) 성능 지표 

Trida-7B-preview 모델은 한국어와 일본어를 대상으로 한 다양한 벤치마크 평가에서 초기 연구 단계임에도 불구하고 상당히 높은 성취를 기록함. 특히 한국어 평가 지표에서 전반적인 성능 향상이 두드러지며, 이는 본 과제에서 구축한 한국어 중심의 데이터 구성과 Diffusion 기반 학습 전략이 효과적으로 작용했음을 보여 줌. 

한국어 지표(KMMLU, KoBEST, HRM8K 등)에서는 기존 AR 기반 모델 대비 전반적으로 높은 수치와 안정적인 성능 흐름이 확인되었으며, 이는 Trida-7B-preview가 지식 회상, 문맥 이해, 상황 판단 등에서 일정 수준 이상의 균형 잡힌 성능을 발휘함을 시사함.
KoBEST와 HRM8K처럼 언어 이해와 실질적 문제 해결 능력을 요구하는 벤치마크에서 높은 점수를 획득한 점은, Diffusion 구조가 한국어 문장 구조의 다양성과 복잡성을 처리하는 데 강점을 갖는다는 근거를 마련한 것으로 볼 수 있음. 

또한 Global-MMLU-ko 및 Global-MMLU-jp에서 확인된 성능은, 한국어 중심으로 개발한 모델임에도 일본어와 같은 유사 언어권으로의 자연스러운 확장 가능성을 확인하는 데 중요한 지표로 작용함. 일본어 평가에서도 비교 모델을 일정 부분 상회하는 결과가 나타났으며, 이는 Diffusion 기반 학습의 특성이 언어 간 구조적 공통점을 활용하는 데 유리하게 작동했음을 보여 줌. 

Koifeval과 같은 생성 품질 평가에서도 높은 점수를 기록함으로써, Trida-7B-preview가 단순한 이해 능력뿐 아니라 문장 표현·자연스러움·문법적 안정성에서 경쟁력을 갖추고 있음을 확인함. 이는 문맥 전체를 기반으로 한 복원 방식이 형식적 정확성과 표현의 자연스러움에 긍정적으로 작용한 것으로 해석됨. 

 

결과적으로 Trida-7B-preview는 한국어뿐 아니라 일본어 평가에서도 의미 있는 성과를 도출하며, 이는 향후 멀티언어·멀티모달 확장 연구에 있어 중요한 전진이며, 확산 기반 파운데이션 모델의 활용 가능성을 대폭 넓히는 기초 성과로 판단됨. 

 

- 추론(inference) 속도 지표 

Trida-7B-preview는 다양한 벤치마크에서 Batch Size 증가에 따른 추론 속도(Token/sec)의 뚜렷한 향상이 확인되었음.
Ifeval, KoIfeval, GSM8K, humaneval 등 전형적으로 응답 길이가 길고 연산 요구가 높은 작업에서도 Batch Size 1 대비 Batch Size 4에서 약 2~3배 가까운 속도 증가가 나타났으며, 이는 Diffusion LLM 구조가 병렬 생성과 고속 디코딩에서 강점을 가진다는 점을 실증적으로 보여줌. 

특히 humaneval과 같은 코드 생성 벤치마크에서는 Batch Size 4 기준 822 Token/sec로 가장 높은 처리 속도를 기록하여, 실시간 응답이 중요한 서비스 분야에서 활용 가능성이 높음을 확인함.
이와 같은 결과는 Trida-7B-preview가 초기 모델임에도 불구하고 경량·고속 추론이 가능한 모델 구조를 성공적으로 구현했다는 의미 있는 성과로 평가됨.

다음과 같은 핵심 요소가 모두 포함된 기능적 레포지토리를 공개함: 

● Masked Decoding Model(MDM) 기반 샘플링 알고리즘 

● Block-wise 병렬 생성 구조(32-token block 병렬화) 

● lm-evaluation-harness 연동 평가 스크립트 

● CLI 및 Gradio 기반 인터랙티브 챗봇 UI 

● vLLM·FlashAttention·멀티모달 확장 모듈 지원 

● 실제 추론·벤치마크 실행이 가능한 완전한 inference pipeline 제공 

 

이를 통해 외부 연구자와 기업은 Diffusion LLM 기반 서비스를 즉시 실험해보고, 속도·성능·추론 구조를 기존 AR 모델과 비교 실험할 수 있는 환경을 확보하게 되었음. 

Repository: https://github.com/trillion-labs/Fast-dLLM-Trida 

 

 

 

 ㅇ 기대효과  

 

 

◆ 학술적 / 기술적 기대효과 

● 확산 기반 + 강화사전학습 조합에 대한 국내 연구 역량 확보
본 과제를 통해 한국어 중심의 Diffusion LLM 연구 환경을 구축함으로써, 기존에는 해외 연구에 의존하던 “언어용 Diffusion + RL-pretraining” 분야에서 국산 독자 기술 기반을 마련하게 됨.
향후 국내 대학·연구기관과의 공동 연구 또는 후속 논문 제출을 통해, 한국어 및 동아시아 언어권에 맞춘 Diffusion-RL LLM 생태계가 본격적으로 성장할 것으로 기대할 수 있음. 

 

● 고속 추론 + 병렬 생성 가능성 확인 → 새로운 연구 패러다임 제시
전통적인 autoregressive 모델은 토큰을 순차 생성해야 하므로 긴 문장·대용량 생성 시 latency와 비용 문제가 있었지만, Diffusion LLM은 병렬 생성 구조와 denoising 과정을 통해 고속 생성 가능성을 제시.
이로써 “속도와 효율성, 유연한 생성 방식”을 모두 만족하는 언어모델 구조에 대한 실증이 이루어졌으며, 이는 국내외 연구자들 간의 후속 연구 촉매가 될 수 있음 

 

● 멀티턴 대화, 조건 생성, 제어 가능성 등 향후 확장 연구 기반 마련
Diffusion-LM 계열 연구들은 제어(controlled generation), 조건부 생성(conditioned generation), 다양성 있는 생성(diverse generation)에 강점이 있다는 보고가 잇따르고 있음.
본 과제 결과물을 기반으로, 향후 RL 기반 fine-tuning, preference RL, human-in-the-loop 등을 결합한 안정적이고 제어 가능한 한국어 생성 AI를 만들 수 있는 학술적 토대를 마련함. 

 

 

◆ 사업적 / 산업적 기대효과 

● Agent, 특화 AI, 실시간 서비스에서의 경쟁력 확보
고속 추론과 병렬 토큰 생성이 가능한 Diffusion LLM은, 기존 AR 기반 모델이 지니던 latency·비용 제약을 크게 완화하게 됨. 이는 특히 대화형 에이전트(agent), 모바일 / 엣지 AI, 실시간 챗봇, 고객 상담, 코드 생성/자동화, 콘텐츠 생성 등 latency·비용이 중요한 응용 분야에서 차별적인 경쟁력을 제공할 수 있게 됨. 

 

● 강화학습(RL)을 통한 고도화된 사용자 맞춤형 및 제어 가능한 AI 서비스 가능성
본 과제에서 설계한 RL-pretraining 구조를 바탕으로, 사용자 피드백 기반 fine-tuning, preference tuning, 안전성 제어 등을 포함한 고품질 맞춤형 AI 서비스를 제공할 수 있는 기반이 마련됨.
이는 단순한 생성 AI를 넘어, 실제 제품화 가능한 서비스 — 예: 기업용 AI 어시스턴트, 맞춤 상담 챗봇, 도메인 특화 생성기 — 로 전환 가능성을 높임. 

 

● 국산 파운데이션 모델 보유를 통한 비용 절감 및 시장 확대 기회
해외 대형 모델을 API 형태로 사용하는 것보다, 자체 개발한 한국어 Diffusion LLM을 바탕으로 하면 추론 비용 절감 및 내재화(in-house) AI 도입이 가능.
이는 스타트업, 중견기업, 공공기관 등 다양한 수요자들에게 AI 도입 문턱을 낮추며, 국내 AI 서비스 시장 전체의 확대를 유도할 수 있음. 

 

● 장기적으로 국내 AI 생태계 및 산업 경쟁력 강화
학계와 산업계가 공통의 기술 토대를 사용하게 됨으로써, 한국어 특화 LLM → 응용 서비스 → 산업화로 이어지는 선순환 구조 구축이 가능해지게 됨.
특히, 향후 멀티모달 확장(any-to-any 모델), 온디바이스 AI, 에지 AI 등에 대한 연구와 사업이 이어진다면, 국내 AI 산업의 자립성과 글로벌 경쟁력이 동시에 강화될 것으로 기대됨. 

 

 

3. 자체평가 

 

◆ Diffusion 모델의 효과적인 개발 

제한된 연산 자원과 촉박한 일정 속에서도, 본 연구팀은 Diffusion LLM 개발의 핵심 구조를 빠르게 파악하고 가장 효율적인 학습 전략을 선택해 안정적인 hero run 달성에 성공했음.
특히 토큰 단위와 블록 단위 Diffusion을 모두 실험하며 구조적 차이를 비교·분석했고, 그 과정에서 Block Diffusion 구조가 한국어 모델링에 특히 유리하다는 근거를 확보함.
이는 기존 AR 모델 대비 고속 추론이 가능하다는 구조적 장점을 실증한 것이며, 본 연구가 단순 실험이 아니라 국내 최초로 Diffusion LLM 실용화를 위한 실제 기술적 토대를 마련한 성과라고 평가할 수 있음. 

또한 학습 안정성, scaling 특성, 학습 효율 등을 종합적으로 검증하면서 Diffusion 기반 LLM이 향후 한국어·멀티모달 모델에서 충분히 경쟁력이 있음을 확인하였고, 차세대 파운데이션 모델의 한 축으로 발전시킬 수 있다는 가능성을 명확히 보여줌. 

 

 

◆ 새로운 연구 주제에 대한 효율적 접근 

Diffusion LLM과 RL-pretraining은 국제적으로도 연구가 이제 막 본격화되는 새로운 주제이며, 국내에는 선행 연구·참고 사례가 거의 없는 상황이었음.
그럼에도 불구하고 본 연구팀은 학습 구조를 빠르게 해석하고 실험 환경을 직접 구축하여 짧은 시간 안에 학습·평가·비교 실험까지 완주하는 성과를 냈음. 

Diffusion 모델의 생성 메커니즘, 블록 단위 학습 방식, denoising 스케줄링, 효율적인 batch 구성, 병렬화 전략 등 새로운 기법이 난이도 높은 주제였음에도, 해당 내용들을 빠르게 흡수하고 실제 모델 개발로 연결해 냄으로써, 국내에 부족했던 Diffusion LLM 연구 기반을 단기간에 확보하는 데 성공함. 

이 과정은 단순히 모델 하나를 만들었다는 의미를 넘어서, 앞으로 한국에서도 Diffusion 기반 연구가 본격적으로 진행될 수 있는 내재적 연구 역량을 확보했다는 점에서 매우 의미가 큼. 

 

 

◆ 국내 GPU 자원의 효과적인 활용 

본 연구는 NIPA를 포함한 국내 GPU 클라우드 자원만으로 대규모 Diffusion LLM을 성공적으로 학습했다는 점에서도 높은 평가가 가능함.
Diffusion LLM은 일반 AR 대비 병렬화가 까다롭고, 학습 안정성을 유지하기 위해 세심한 자원 관리가 필요하지만, 제한된 GPU 수량 속에서도 데이터 병렬화·텐서 병렬화·컨텍스트 병렬화 등 다양한 전략을 조합해 최적의 학습 효율을 만드는 데 성공함. 

특히 hero run 기준으로 안정적으로 수렴하는 세팅을 찾아낸 것은 단순 노력이 아니라 국내 GPU 인프라 환경에서 대규모 모델을 어떻게 운영해야 하는지에 대한 실질적 경험을 확보한 것임.
이는 향후 더 많은 GPU 자원이 확보될 경우 모델 규모 확대, 멀티모달 확장, RL 실험 가속 등으로 바로 이어질 수 있는 확장성을 의미함. 

 

 

4. 계획대비 부진사업 현황 및 대책  

 

1) RL 심화 연구의 일정 관련 사항 

● 본 과제에서는 Diffusion LLM의 전체 구조를 안정적으로 구축하기 위해, 가장 가능성이 높은 학습 조합을 선택하여 집중적으로 학습을 진행하고 모델이 실제로 완전하게 수렴하도록 만드는 데 우선순위를 두었음. 

● 이러한 핵심 학습(run)을 성공적으로 마무리하는 과정에서 RL(RPT) 심층 실험의 일정은 다소 후순위로 조정됨. 

● 이는 모델 전체의 품질과 신뢰도를 확보하기 위해 우선순위 조정으로 추후 RL 연구를 더욱 정교하게 확장하여 개선하고자 함.  

 

대책:
사업 종료 이후 RL 모듈을 별도 확장 과제로 다루어, 이미 확보한 파이프라인 위에서 보상 설계·정책 탐색·튜닝을 체계적으로 심화하는 후속 연구를 이어갈 예정임. 

 

2) GPU 자원 규모에 따른 실험 범위 조정 

● 제한된 GPU 자원 내에서 실험의 효율성을 극대화하기 위해, 가장 유망한 모델 구조와 설정을 우선적으로 선택해 충분한 학습을 진행하고 실제 사용 가능한 수준까지 품질을 끌어올리는 데 연구를 집중함. 

● 이 과정에서 멀티모달 확장 및 any-to-any 모델 실험은 개념 검토와 초기 실험에 초점을 맞추는 방향으로 조정됨. 

● 이는 단기적으로는 범위가 축소된 것처럼 보이지만, 장기적으로는 확장 실험을 위한 방향성과 기술적 기반을 확보했다는 점에서 의미가 큼. 

 

대책:
후속 연구에서는 GPU 자원을 확대하여 멀티모달 Diffusion LLM 실험을 본격화하고, Diffusion 구조의 장점을 극대화할 수 있는 범용 모델 연구를 단계적으로 추진할 예정임. 

 

 

5. 향후 계획 

 

◆ 오픈소스 공개를 통한 학계 연계 및 연구 지속 

본 과제에서 개발한 모델 및 학습 파이프라인의 핵심 구성요소를 순차적으로 오픈소스 형태로 공개하여, 국내·외 연구자들이 동일 환경에서 재현·확장 연구를 수행할 수 있도록 지원할 계획임. 

특히 Diffusion LLM, Block Diffusion, RPT(RL Pre-training) 등은 국내에서는 선행 사례가 드문 연구 분야이므로, 이를 공개함으로써 다음과 같은 효과를 기대함: 

● 대학·연구소와의 공동연구 활성화 

● Diffusion 기반 언어모델의 대규모 벤치마크 축적 

● 후속 연구자들이 모델 확장·압축·fine-tuning·task별 발전 연구를 수행할 수 있는 기반 마련 

● 공개된 코드/데이터 subset/API가 학계의 실험 표준으로 자리잡아 국내 연구 생태계의 품질 향상 

 

또한 프로젝트 종료 후에도 지속적인 버전 업데이트·성능 개선·학습 기록 관리 등을 통해 연구결과물이 일회성에 그치지 않고 지속 가능한 연구 플랫폼으로 유지·고도화될 수 있도록 관리 체계를 구축할 예정임. 

 

◆ Diffusion 모델을 활용한 데이터 효율적인 특화 모델의 사업화 

본 과제에서 확보한 Diffusion LLM 및 강화사전학습 기반 모델 기술을 활용하여, 산업별 특화 AI 모델 사업화를 본격적으로 추진할 계획임.
Diffusion 구조의 고속 추론 특성·낮은 비용·병렬 생성 가능성은 특정 산업군에서 특히 경쟁력을 가질 것으로 판단됨. 

1. 산업별 특화 모델 개발 및 제공 

● 금융, 헬스케어, 교육, 커머스 등 분야별 데이터 특성을 반영한 도메인 특화 Diffusion LLM을 개발할 예정임. 

● AR 기반 파운데이션 모델 대비 추론 비용·응답 지연이 크게 감소하므로, 실시간성이 요구되는 상담, 문서처리, 정보검색 시스템에서 높은 효율성을 제공할 수 있음. 

 

2. Agent·자동화 솔루션으로의 사업 확장 

◆ 본 과제에서 확인된 Diffusion 기반 고속 추론 성능을 활용하여, ‘모바일 에이전트’, ‘업무자동화 에이전트’, ‘고객 응대 에이전트’ 등 다양한 agent 서비스로 확장할 예정임. 

◆ 경량화 및 온디바이스 적용 가능성이 높아 모바일 환경에서도 상용 적용이 용이함. 

 

3. 해외 언어권·멀티모달 시장 확장 

● 한국어 중심 Diffusion LLM을 기반으로, 일본어 등 아시아 언어권으로 확장 계획임. 

● 향후 멀티모달 a2a(any-to-any) Diffusion 구조까지 연구 범위를 확대해, 영상·음성·텍스트를 통합한 산업 서비스로 확장할 가능성을 확보함.
